# ffx - FFmpeg, but improved

A beautiful, intuitive wrapper for ffmpeg with smart defaults and helpful guidance.

## Installation

```bash
pip install rich click humanize
# Save ffx.py as 'ffx'
chmod +x ffx
# Move to /usr/local/bin/ or add to PATH
```

## Requirements

- ffmpeg installed on system
- Python 3.7+

## Usage

```bash
ffx --help
```
